# Admin Backoffice (Electron)

Wrapper khusus admin/backoffice.

## Run

PowerShell:
```powershell
$env:ADMIN_SERVER_URL="http://192.168.88.245/ritel4/public/admin/login.php"
npm install
npm run start
```

## Build EXE

```powershell
$env:ADMIN_SERVER_URL="http://192.168.88.245/ritel4/public/admin/login.php"
npm run build
```

Hasil build ada di folder `dist`.
