# POS Kiosk (Electron)

Wrapper khusus POS kasir.

## Run

PowerShell:
```powershell
$env:POS_SERVER_URL="http://192.168.88.245/ritel4/public/POS/login.php"
npm install
npm run start
```

## Build EXE

```powershell
$env:POS_SERVER_URL="http://192.168.88.245/ritel4/public/POS/login.php"
npm run build
```

Hasil build ada di folder `dist`.
